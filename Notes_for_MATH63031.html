1.1 Axiomx for a Field¶

A-1. Closure property. If $a$ and $b$ are numbers, there is one and only one number, denoted $a+b$, called their sum.
A-2. Commutative law. For any two numbers $a$ and $b$, the equality $$b+a=a+b$$ holds.
A-3. Associative law. For all numbers $a$, $b$, and $c$, the equality $$(a+b)+c=a+(b+c)$$ holds.
A-4. Existence of a zero. There is one and only one number zero, called zero, such that $a+0=a$ for any number $a$. (Add commentary)
A-5. Existence of a negative. If $a$ is any number, there is one and only one number $x$ such that $a+x=0$. This number is called the negative of $a$ and is denoted by $-a$.

Theorem 1.1. If $a$ and $b$ are any numbers, then there is one and only one number $x$ such that $a+x=b$. This number $x$ is given by $x=b+(-a)$.
PROOF. We must establish two results (i) that $b+(-a)$ satisfies the equation $a+x=b$ and (ii) that no other number satisfies this equation. To prove (i), suppose that $x=b+(-a)$. Then, using Axioms A-2 through A-4 we see that $a+x=a+[b+(-a)]=a+[(-a)+b]=[a+(-a)]+b=0+b=b$. Therefore (i) holds. To prove (ii), suppose that $x$ is some number such that $a+x=b$. Adding $(-a)$ to both sides of this equation, we find that $$(a+x)+(-a)=b+(-a).$$ Now, $$(a+x)+(-a)=a+[x+(-a)]=a+[(-a)+x]=[a+(-a)]+x=0+x=x.$$ We conclude that $x=b+(-a)$, and the uniqueness of the solution is established. $\square$

Notation. The number $b+(-a)$ is denoted by $b-a$.

     Axioms of Multiplication and Division

M-1. Closure property. If $a$ and $b$ are numbers, there is one and only one number, denoted by $ab$ (or $a*b$), called their product.
M-2. Commutative law. For every two number $a$ and $b$, the equality $$ba=ab$$ holds.
M-3. Associative law. For all numbers $a$, $b$, and $c$, the equality $$(ab)c=a(bc)$$ holds.
M-4. Existence of a unit. There is one and only one number $u$, different from zero, such that $au=a$ for every every number $a$. This number $u$ is called the unit and (as is customary) is denoted by 1.
M-5. Existence of a reciprocal. For each number a different from zero there is one and only one number $x$ such that $ax=1$. This number $x$ is called the reciprocal of $a$ (or the inverse of $a$) and is denoted by $a^{-1}$ (or $1/a$).

Theorem 1.3. If $a$ is any number, then $a*0=0$.
PROOF. Let $b$ be any number. Then $b+0=b$, and therefore $a(b+0)=ab$. From the distributive law (Axiom D), we find that $$(ab)+(a*0)=(ab),$$ so that $a*0=0$ by Axiom A-4. $\square$

Theorem 1.5. (i) We have $ab=0$ if and only if $a=0$ or $b=0$ or both.
(ii) We have $a\ne0$ and $b\ne0$ if and only if $ab\ne0$.
PROOF. We must prove two statements in each of Parts (i) and (ii). To prove (i), observe that if $a=0$ or $b=0$ or both, then it follows from Theorem 1.3 that $ab=0$. Going the other way in (i), suppose that $ab=0$. Then there are two cases: either $a=0$ or $a\ne0$. If $a=0$, the result follows. If $a\ne0$, then we see that $$b=1*b=(a^{-1}a)b=a^{-1}(ab)=a^{-1}*0=0.$$ Hence $b=0$ and (i) is established. To prove (ii), first suppose $a\ne0$ and $b\ne0$. Then $ab\ne0$, because $a\ne0$ and $b\ne0$ is the negation of the statement "$a=0$ or $b=0$ or both." Thus (i) applies. For the second part of (ii), suppose $ab\ne0$. Then $a\ne0$ and $b\ne0$ for, if one of them were zero, Theorem 1.3 would apply to give $ab=0$. $\square$

1.2 Natural Numbers and Sequences¶

(Add commentary)

Definitions. The set (or collection) of all real numbers is denoted by $\R^{1}$. The set of ordered pairs of real numbers is denoted by $\R^{2}$, the set of ordered triples by $\R^{3}$ and so on.

  • Relation: A relation from $\R^{1}$ to $\R^{1}$ is a set of ordered pairs of real numbers, that is, a relation from $\R^{1}$ to $\R^{1}$ is a set in $\R^{2}$.
  • Domain: The domain of this relation is the set in $\R^{1}$ consisting of all the first elements in the ordered pairs.
  • Range: The range of the relation is the set of all the second elements in the ordered pairs. Observe that the range is also a set in $\R^{1}$.
  • Function: A function $f$ from $\R^{1}$ into $\R^{1}$ is a relation in which no two ordered pairs have the same first element. We use the notation $f: \R^{1} \rightarrow \R^{1}$ for such a function.
    • Mapping: The word mapping is a synonym for function. If $D$ is the domain of $f$ and $S$ is its range, we shall sometimes use the notation $f: D \rightarrow S$. A function is a relation (set of $\R^{2}$) such that for each element $x$ in the domain there is precisely one element $y$ in the range such that the pair $(x,y)$ is one of the ordered pairs which constitute the function. Occassionally, a function will be indicated by writing $f: x \rightarrow y$. Also, for a given function $f$, the unique number in the range corresponding to an element $x$ in the domain is written $f(x)$. The symbol $x \rightarrow f(x)$ is used for this relationship. We assume that the reader is familiar with functionL notation.
  • Sequence: A sequence is a function which has as its domain some or all of the natural numbers. If the domain consists of finite number of positive integers, we say the sequence is finite. Otherwise, the sequence is called infinite.

Proposition 1.1 (Sum). If $a_1,a_2,...,a_n$ is given sequence, then there is a unique finite sequence $b_1,b_2,...,b_n$ with the properties: $$b_1=a_1$$ and, in general, $$b_{i+1}=b_i+a_{i+1} \text{ for } i=1,2,...,n-1 \text{ if } n > 1.$$ Proposition 1.2 (Product) If $a_1,...,a_n$ is given sequence, then there is a unique finite sequence $c_1,...,c_n$ with the properties: $$c_1=a_1$$ and, in general, $$c_{i+1}=c_i*a_{i+1} \text{ for } i=1,2,...,n-1 \text{ if } n > 1.$$ Definitions. For all integers $n \geqslant 1$, the sum and product of the numbers $a_1,a_2,...,a_n$ are defined by $$a_1+a_2+ \cdots +a_n \equiv b_n \text{ and } a_1*a_2* \cdots *a_n \equiv c_n$$ We use the notation $$\sum_{i=1}^{n} a_i=a_1+a_2+ \cdots +a_n \text{ and } \prod_{i=1}^{n} a_i = a_1*a_2* \cdots *a_n.$$ Proposition 1.3. If $a_1,a_2,...,a_n,a_{n+1}$ is any sequence, then $$\sum_{i=1}^{n+1} a_i = \left( \sum_{i=1}^{n} a_i \right)+a_{n+1} \text{ and } \prod_{i=1}^{n+1} a_i = \left( \prod_{i=1}^{n} a_i \right)*a_n.$$ Proposition 1.4. If $a_1,a_2,...,a_m,a_{m+1},...,a_{m+n}$ is any sequence, then $$\sum_{i=1}^{m+n} a_i = \sum_{i=1}^{m} a_i + \sum_{i=m+1}^{m+n} a_i$$ and $$\prod_{i=1}^{m+n} a_i = \prod_{i=1}^{m} a_i + \prod_{i=m+1}^{m+n} a_i$$

Proposition 1.5. The sum of a finite sequence can be obtained by separating the given sequence into several shorter sequences, adding the terms in each of these, and then adding the results. A similar statement holds for products.
Proposition 1.6. The sum of a given finite sequence is independent of the order of its terms. The product of a funeute sequence is independent of thr order of its terms.
Proposition 1.7. If $a$ is any number and $b_1,b_2,...,b_n$ is any sequence, then $$a*(b_1+b_2+ \cdots + b_n)=(b_1+b_2+ \cdots + b_n)*a=ab_1+ab_2+ \cdots +ab_n.$$

Definitions. A real number is an integer if and only if it is either zero, a natural number, or the negative of a natural number.

  • A real number $r$ is said to be a rational if and only if there are integers $p$ and $q$, with $q \ne 0$, such that $r=p/q$.
  • A real number which is not rational is called irrational.

Definition 1. A field consists of a set of k and two binary operations "+" and "." destined on k for which the following conditions are satisfied:
(i) $(a+b)+c=a+(b+c)$ and $(a*b)*c=a(b*c)$ for all $a$,$b$,$c\in k$ (associativity).
(ii) $a+b=b+a$ and $a*b=b*a$ for all $a,b,c \in k$ (commutativity).
(iii) $a*(b+c)=a*b+a*c$ for all $a,b,c \in k$ (distributivity).
(iv) There are $0$,$1 \in k$ such that $a+0=a*1=a$ for all $a \in k$ (identities).
(v) Given $a \in k$, there is $b \in k$ such that $a+b=0$ (additive inverses).
(vi) Given $a \in k$, $a\ne0$, there is $c \in k$ such that $a*c=1$ (multiplicative inverses).

Remarks: The fields most commonly used in the text are $\Q$, $\R$, and $\C$. In the exercises to $\S$1 of Chapter 1, we mention the field $\F_2$ which consists of the two elements 0 and 1. Some more complicated fields are discussed in the text. For example, in $\S$3 of Chapter 1, we define the field $k(t_1,...,t_m)$ of rational functions in $t_1,...,t_m$ with coefficients in $k$. Also, in $\S$5 of Chapter 5, we introduce the field $k(V)$ of rational functions on an irreducible variety $V$.
